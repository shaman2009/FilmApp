FilmApp
=======

###For Dream

- 新浪微博登录
- 输入电影名称 搜索电影
- 跳转到图片选择界面（符合搜索结果的电影海报，用户可以自己选择）
- 加上自己想要说的话，然后点击“发布”
- @好友自动补全
